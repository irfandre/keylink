KeyLink Navigation

In google search result keypress any alphanumeric(0-9a-z) value which is associated with links to visit that link

Use / to focus on first input field.

Use (comma) , / . (period) for scrolling in all websites.

Use J / Z and K / W keys to scroll down and up respectively on all website except google.

Use Option + - to switch to previously visited tab.

Note: Some website use custom script for scrolling longpress scrolling might slow on those websites.
